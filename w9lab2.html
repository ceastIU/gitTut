SQL Questions

  • A primary key is a unique value that represents a whole row.
  • A foreign key is a value that refers to another table's primary key.
  • The difference between a tuple and an attribute is that a tuple represents the rows, where an attribute represents the columns.
  • SQL stands for 'Structured Query Language' and it represents short, but powerful lines of code for manipulating databases.
  • Primary key example could be social sercurity number or driver's license number as these are unique for each person
